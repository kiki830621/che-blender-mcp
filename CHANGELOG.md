# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-23

### Added

- Initial release
- **execute_script**: Execute arbitrary Python code in Blender
- **list_objects**: List all scene objects with basic info
- **get_object_info**: Get detailed information about specific objects
- **render**: Render scene and return base64-encoded image
- **get_scene_info**: Get comprehensive scene information
- **screenshot**: Capture viewport screenshot
- Support for Cycles and EEVEE render engines
- Configurable Blender path via `BLENDER_PATH` environment variable
- mcpb package manifest for easy distribution
