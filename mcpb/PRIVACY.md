# Privacy Policy

## Data Collection

This MCP server:
- Runs entirely locally on your Mac
- Does not collect or transmit any personal data
- Does not connect to external servers
- Only accesses local Blender installation and specified blend files

## Data Storage

No data is stored by this extension beyond what is necessary for its operation.
Temporary render files are created in the system temp directory and deleted after use.

## Third-Party Services

This extension does not use any third-party services.
It interacts only with locally installed Blender.
